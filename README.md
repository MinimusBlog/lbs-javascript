# Учебное Web-приложение для системы технической поддержки.
Веб-пирложение реализовано на чистом js.

## Функциональность
- просмотр всех тикетов
- фильтрация новых тикетов по статусу
- статистика конкретизирована
- кнопка перехода на страницу создания обращения
- сортировка по значению (value) по убыванию
- Удаление обращения из списка
- кнопка перехода на создание обращения

## Технологии
- HTML
- CSS (fonts Roboto)
- JavaScript

## Структура
- `index.html` - главная страница
- `style.css` - стили
- `data.js` - массив объектов
- `logic.js` - функции обработки данных
- `script.js` - работа с DOM
- `pages/forms-ticket.html` - страница создания обращения

## Данные 
Встроенные массивы `tickets` и `users` иммитируют базу-данных (БД) из ЛР2.
В ЛР 3 массив `items` в data.js имитирует БД
